github搭建hexo静态网站(windows64位)

准备工作

  • git
  • node.js
  • hexo

hexo搭建步骤

  1. 安装git
  2. 安装node.js
  3. 安装hexo
  4. 使用hexo
  5. 生成ssh添加到github
  6. 将hexo部署到github
  7. 发布文章

    1. 安装git

  • 下载git
  • 安装git
  • 出现git bash说明安装成功

2. 安装node.js

  • 下载node.js
  • 安装node.js
  • 在cmd窗口中输入npm -v，若不报错，说明安装成功

3. 安装hexo

  • npm install -g hexo-cli

  • 新建完成后，指定文件夹目录下有：

    • node_modules: 依赖包
    • public：存放生成的页面
    • scaffolds：生成文章的一些模板
    • source：用来存放你的文章
    • themes：主题
    • ** _config.yml: 博客的配置文件**

  • 生成静态文件

  • hexo g

  • 访问github方法

  • 进入网站https://github.com.ipaddress.com/www.github.com获取github最新地址

  • 修改hosts文件

5. 生成ssh添加到github

  • 配置git全局环境
  • git config –global user.name “yourname”
  • git config –global user.email “youremail”
  • 检查是否配置成功
  • git config user.name
  • git config user.email
  • 生成ssh公钥，公钥文件一般在用户名文件夹下，文件夹名称为.ssh
  • ssh-keygen -t rsa -C “youremail”
  • 将id_rsa.pub添加到github上
  • 本地使用git bash输入ssh -T git@github.com即可判断是否配置成功

6. 将hexo部署到github

  • 打开hexo配置文件_config.yml,翻到最后，修改为
    YourgithubName就是你的GitHub账户

  • 这个时候需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub

  • 然后

  • 其中 hexo clean清除了你之前生成的东西，也可以不加。

  • hexo generate顾名思义，生成静态文章，可以用hexo g缩写

  • hexo deploy部署文章，可以用hexo d`缩写
